5. Преобразование матрицы чисел в список чисел. [#110228]

    В этом задании нужно преобразовать числовую матрицу в список.
    Дана произвольная матрица

    List<List<Integer>> matrix = List.of(
            List.of(1, 2),
            List.of(3, 4)
    );

    Для того, чтобы преобразовать список списков нужно использовать flatMap.
    flatMap принимает элемент потока и преобразовывает его в новый поток.
    В нашем примере в качестве элементы получается список List.of(1, 2)
    У него можно сразу вызвать метод stream().

    System.out.println(
            matrix.stream().flatMap(e -> e.stream()).collect(Collectors.toList())
    );

    В конце мы можем сразу собрать все элементы в список.
    flatMap(e -> e.stream()) - в этом место можно заменить лямбду на ссылочный метод.
    flatMap(List::stream)

    Задание.
    1. Создайте матрицу Integer[][].
    2. Напишите преобразование этой матрицы в список чисел.
    3. При решении этой задачи нужно использовать методы flatMap, collect, Stream.of
